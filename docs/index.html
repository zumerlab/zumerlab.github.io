We make the kind of libraries we wish already existed — radial layouts, zoom-driven UIs, pixel-perfect DOM capture.

Zumerlab is an independent software studio. We work in the open: every primitive ships with source, docs, and a live demo. No frameworks of frameworks, no vendor lock-in.

Below: the active specimens. Each is in production use, MIT-licensed, and maintained.

§ 01 / About

Built by people who treat the browser as a research surface.

Premise

Most of what the web platform can do is still under-explored. There are angles, depths, and capture surfaces that designers reach for every week and re-implement from scratch.

Zumerlab focuses on those gaps. Small, sharp tools that ship with no transitive dependencies and a permissive license.

  • Open-source by default — MIT, public from day one
  • Zero or near-zero runtime dependencies
  • Live demos, not just screenshots
  • Framework-agnostic; works with plain HTML
  • Designed in the open on GitHub
§ 02 / Specimens

Three primitives, actively maintained, in production.

SPC001

Orbit

CSS framework / radial UIs
@zumer/orbit MIT

A CSS framework for building radial interfaces — gauges, donuts, knobs, pie menus, dashboards — using only CSS. No layout JavaScript.

Drop in two files. Works with plain HTML and any framework (React, Vue, Svelte). Angles, radii, and positions are computed in stylesheets.

Live docs Source npm
SPC002

Zumly

JS library / Z over XY
zumly MIT

Hierarchical zoom navigation: you move in Z (depth) through discrete views laid out in the XY plane. Spatial transitions instead of flat screen swaps.

Pluggable transition drivers (CSS, WAAPI, Anime.js, GSAP, Motion). Plugin API, hash router, prefetch cache. UI-agnostic.

Live demo Source npm
SPC003

snapDOM

Utility / DOM-to-image capture
@zumer/snapdom MIT

Captures any HTML element as a scalable SVG image — preserving styles, fonts, background images, pseudo-elements, and shadow DOM.

Exports to SVG, PNG, JPG, WebP, canvas, or Blob. Supports same-origin iframes, CSS counters, line-clamp. Zero dependencies.

Live demo Source npm
